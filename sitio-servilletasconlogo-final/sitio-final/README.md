# ServilletasconLogo.com.mx

## Estructura
```
/index.html                          → servilletasconlogo.com.mx/
/portavasos-con-logo/index.html      → servilletasconlogo.com.mx/portavasos-con-logo/
/sitemap.xml
/robots.txt
/CNAME
```

## Deploy en GitHub Pages
1. Crear repo público en GitHub
2. Subir todos estos archivos
3. Settings → Pages → Branch: main → / (root)
4. Custom domain: servilletasconlogo.com.mx
5. Activar "Enforce HTTPS"
